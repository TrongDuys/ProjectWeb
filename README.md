# Installing

---

```
clone source code: https://github.com/TrongDuys/ProjectWeb.git
```

# Install Package BackEnd

```
npm install
```

# Install Package FontEnd

```
cd client
```

```
npm install
```

# Run server

- At the root directory

```
npm run server
```

# Run client

- Move folder client

```
cd client
```

```
npm start
```
